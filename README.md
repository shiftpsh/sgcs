### 무료 소프트웨어 및 서비스

| **[학생용 Microsoft Azure 스타터](https://azure.microsoft.com/ko-kr/offers/ms-azr-0144p/)**<br />학교 메일로 신청 | - Azure Cloud Service                                        |
| **[캠퍼스 클라우드](https://scloud.sogang.ac.kr)**<br />SAINT ID로 로그인 | - MATLAB (웹탐색기 > 공통소프트웨어)                         |
